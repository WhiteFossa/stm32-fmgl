void FMGL_API_RenderOneLineDumb ( FMGL_API_DriverContext *  context,
FMGL_API_FontSettings *  fontSettings,
uint16_t  x,
uint16_t  y,
uint16_t *  width,
bool  isDryRun,
char *  string 
)

Draws one line without wrapping at spaces. Newlines aren't allowed, in case of newline L2HAL_Error(L2HAL_ERROR_WRONG_ARGUMENT) will be called. Draws until line end or until no pixels of next character falls into screen area. Returns (in width parameter) width of drawn line.

Parameters
contextPointer to FMGL library context.
fontSettingsPointer to font settings.
x,yTop left text coordinates.
widthPointer to variable, where rendered text width will be stored.
isDryRunIf true, then doesn't draw anything, just calculating width.
stringText to render.

FMGL_API_XBMTransparencyMode  transparency 
)

Renders XBM image such way, that (0,0) pixel of image will be placed at (x,y). Image is being scaled up by scaleX and scaleY. XBM's active pixels will be displayed using activeColor, inactive - using inactiveColor.

Parameters
contextDriver context.
imagePointer to XBM image struct.
xX coordinate of top left image corner.
yY coordinate of top left image corner.
scaleXScale image in horizontal direction by this factor.
scaleYScale image in vertical diraction by this factor.
activeColorDraw image active pixels with this color.
inactiveColorDraw image inactive pixels with this color.
transparencyTransparency mode.
